# 配置

## webpack 配置

在项目根目录下，我们可以看到 `.env.local` 文件,在这里我们可以配置 `webpack-plugin-cep-link`

``` js
# @kuaizi/vue-cli-plugin-cep
VUE_APP_CEP_ID=ai.kuaizi.ps.cep
VUE_APP_CEP_DEBUG=true
VUE_APP_CEP_PORT=9000
```
